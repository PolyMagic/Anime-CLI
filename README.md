![Preview](https://i.imgur.com/CfebORF.png)

# Instalation

Install Node form nodejs.org (Version 10 is recommended)

Run this command in terminal.

```
npm i -g anime-cli
```

If you are on linux you need to use sudo for this command;

# Usage

Run

```
anime-cli
```
